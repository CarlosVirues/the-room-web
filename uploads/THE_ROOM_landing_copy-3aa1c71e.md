## NOTAS TÉCNICAS PARA EL BUILD

**Stack recomendado:**
- Next.js + Tailwind + Framer Motion (si quieres animaciones en hero)
- HTML + Tailwind puro + Cloudflare Pages (deploy en 1 hora)

**Tipografía:**
- Body: Inter o Söhne
- Stats grandes / mono: JetBrains Mono o Suisse Int'l Mono

**Paleta:**
- Negro profundo (#0A0A0A)
- Rojo THE ROOM (#E63946 o equivalente del deck)
- Crema/off-white para contraste (#F5F1EB)

**Hero background:**
Dashboard animado tipo data wall con menciones rotando en tiempo real. Tono editorial-político, no SaaS startup.

**Reglas de diseño:**
- Sin emojis
- Sin íconos animados Lottie
- Tipografía grande, mucho espacio en blanco
- Tablas y datos en mono para sensación editorial / Bloomberg
